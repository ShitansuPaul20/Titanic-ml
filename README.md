# Titanic Survival Analysis 🚢

An end-to-end Machine Learning project analyzing Titanic passenger survival data.

## Live Demo
https://titanic-ml-tdepamur6ehxakc3apugun.streamlit.app/

## Project Structure
- `Tytanic.ipynb` - Data Preprocessing and EDA
- `Tytanic2.ipynb` - ML Models (Logistic Regression + Random Forest)
- `app.py` - Interactive Streamlit Dashboard

## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis with visualizations
- Two ML models compared: Logistic Regression (79.89%) and Random Forest (82.68%)
- Interactive survival prediction tool

## Tech Stack
Python, Pandas, NumPy, Seaborn, Matplotlib, Scikit-learn, Streamlit
